libqrencode
===========

C语言生成二维码的库，关于介绍，请参考
https://github.com/fukuchi/libqrencode

编译：
使用vs2010环境
该库依赖libpng
需要编译\lpng1615\projects\vstudio\vstudio.sln生成zlib和pnglib

使用：
打开libqrencode.sln
参考test.cpp

说明：
基于libqrencode的qrencode库，port了最新的3.9.0版本
